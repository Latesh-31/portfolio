# Latesh K — Portfolio

Personal portfolio site built with Tailwind utilities and vanilla JavaScript to showcase Machine Learning and Full-Stack projects.

## Overview

This repository contains a single-page portfolio with an organic/earthy visual style, animated intro, custom cursor, particle background, scroll reveal, and 3D-tilt project cards.

Files:
- `index.html` — main HTML file
- `styles.css` — CSS (theme variables, animations, layout)
- `script.js` — JavaScript for animations and interactions
- `LICENSE` — MIT License
- `.gitignore` — typical web project ignores

## Preview
Repository: https://github.com/Latesh-31/portfolio

To preview locally, open `index.html` in a browser, or serve with a static server:

```bash
# from project root
python -m http.server 8000
# then open http://localhost:8000
```

## Enable GitHub Pages
1. Go to `Settings` → `Pages` in the GitHub repository.
2. Select branch `main` and folder `/ (root)`.
3. Save — the site should publish at `https://latesh-31.github.io/portfolio` after a short delay.

## Customization
- Edit CSS variables at the top of `styles.css` to change color theme and typography.
- Replace the profile image background in `index.html` with your own photo.
- Update contact details in the Contact section.
